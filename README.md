### Kanban PubliDoCeu

## 1. Estrutura

- Cada cliente possui um painel (quadro) exclusivo ao fazer login.  
- Colunas padrão para o fluxo de trabalho (Ex.: Planejamento, Em Produção, Revisão, Concluído).  
- Criação e edição de tarefas (cartões).  

## 2. Gestão de Tarefas

- Atribuição de responsáveis (membro da agência).  
- Definição de prazos de entrega.  
- Descrição detalhada da tarefa.  
- Anexos (documentos, imagens, artes, relatórios).  
- Checklists para etapas dentro de cada tarefa.  

## 3. Comunicação

- Comentários dentro das tarefas (cliente e agência podem interagir).  
- Notificações quando uma tarefa é atualizada ou concluída.  

## 4. Acompanhamento

- Filtros e busca por status, responsável ou prazo.  
- Visualização do progresso das tarefas (ex.: quantas estão em produção, quantas concluídas).  
- Relatórios básicos de andamento do projeto (tarefas concluídas, pendentes, em atraso).  

## 5. Experiência do Cliente

- Permissões limitadas para clientes (somente visualizar e comentar).  
- Possibilidade de exportar relatórios em PDF para acompanhamento externo.  

## 6. Extras / Avançados

- Automatizações (ex.: mover automaticamente para “Concluído” quando checklist = 100%). 
- Exportação de dados (CSV, PDF).
